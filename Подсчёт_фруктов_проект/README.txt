Инструкция по запуску:

1. Установите Python.
2. Установите зависимости:
   pip install flask opencv-python numpy ultralytics

3. Убедитесь, что файл модели yolov8n.pt находится в папке /model
   Если он не скачался автоматически, загрузите с:
   https://github.com/ultralytics/assets/releases/download/v0.0.0/yolov8n.pt

4. Запустите проект:
   python app.py

5. Перейдите в браузере по адресу:
   http://localhost:5000
